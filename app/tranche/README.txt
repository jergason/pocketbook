****************************************
           INCLUDED TOOLS
****************************************

Add: Uploads.

Get: Downloads.

GUI: Launches the Java Swing GUI.

Server: Starts a data server.



****************************************
        GENERAL INSTRUCTIONS
****************************************

PREREQUISITE: Install Java 5 or later version

1. Check that ports 443 and 1045 are open.

2. Unzip the downloaded zip file.

3. Navigate to the top-most directory in the unzipped file.

4. Run the following command to print usage, replacing <JAR> with the jar file name: java -jar ./<JAR> --help
